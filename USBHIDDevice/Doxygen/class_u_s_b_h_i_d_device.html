Detailed Description

The USBHIDDevice class.

The USBHIDDevice can communicate with a standard USB HID Device and is optimized to handle a variable device (like an optical joystick) from curdes.

The USBHIDDevice constructor.

The CaptureThreadButtonTriggered Signal.

This signal is generated when the capture thread detects a change in the Buttons. NewValue holds the new value of the Buttons after the change was detected, ChangedBits holds a mask of the Button-bits that changed value.

The CaptureThreadJoystickChanged Signal.

This signal is generated when the capture thread detects a change in the X,Y position of the Joystick. The Xpos and Ypos hold the last value of the Joystick when the change was detected.
